# Tuning Garage v3 — Conteúdo completo (tom entusiasta)

Este projeto contém um site SPA com conteúdo detalhado e tom entusiasta sobre:
- JDM (história, modelos icônicos, práticas de tuning)
- Carros Alemães (filosofia, exemplos e tuning)
- Tunagem Brasileira (cultura, modelos, dicas)
- Peças & Funcionamento (motor, turbo, supercharger, intercooler, ECU, escapamento)

## Como rodar localmente
Recomenda-se executar em um servidor local por causa de módulos ES:
- Com Python:
  python -m http.server
  e abra http://localhost:8000
- Ou use Live Server no VS Code.
